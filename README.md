## 中文說明

為 [Claude Code](https://claude.ai/code) 打造的實用套件集合。

### 套件列表

#### [claude-hearts](./claude-hearts) ✦

可愛的 statusline，顯示用量限制、Context 使用量與本次費用，支援 14 種主題。

```bash
git clone https://github.com/tseng7510/claude_code_plugin.git
cd claude_code_plugin/claude-hearts
node bin/install.js
```

#### [claude-ping](./claude-ping) 🔔

桌面通知套件，當 Claude 完成任務或等待你確認時，自動發送系統通知。支援 macOS。

```bash
git clone https://github.com/tseng7510/claude_code_plugin.git
cd claude_code_plugin/claude-ping
node bin/install.js
```

## English

A collection of plugins for [Claude Code](https://claude.ai/code).

### Packages

#### [claude-hearts](./claude-hearts) ✦

A cute statusline showing rate limit usage, context window, and session cost. Supports 14 themes.

```bash
git clone https://github.com/tseng7510/claude_code_plugin.git
cd claude_code_plugin/claude-hearts
node bin/install.js
```

#### [claude-ping](./claude-ping) 🔔

Desktop notifications for Claude Code. Get notified when Claude finishes a task or needs your input. macOS only.

```bash
git clone https://github.com/tseng7510/claude_code_plugin.git
cd claude_code_plugin/claude-ping
node bin/install.js
```
